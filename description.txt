			Принципы работы программы
	
		1. В конфигурационном файле config.json задаются названия файлов, в которых будет осуществляться поиск.
		2. Программа обходит все файлы и индексирует их так, чтобы по поисковому запросу найти наиболее
	релевантные документы.
		3. Запросы содержатся в файле запросов requests.json. Запрос - это набор слов, по которым ищутся документы.
		4. Запрос трансформируется в список слов.
		5. В индексе ищутся документы, в которых встречаются эти слова.
		6. Результаты поиска ранжируются, сортируются и выводятся в файл answers.json. Максимальное количество
		документов задаётся в конфигурационном файле.
		
			Класс ReadWriteJSONFile
		
		Класс реализует чтение и запись JSON-файлов
		
			Выполняет следующие функции:
		
		1. Читает JSON-файл
		2. Записывает JSON-файл
		3. Хранит JSON-объект
		4. Получает JSON-объект
		5. Устанавливает JSON-объект
		
			Класс ReadWriteTextFile
		
		Класс реализует чтение и запись текстовых файлов
		
			Выполняет следующие функции:
		
		1. Читает текстовый файл
		2. Записывает текстовый файл
		3. Хранит текстовый объект
		4. Получает текстовый объект
		5. Устанавливает текстовый объект
		
		
			Класс ConverterJSON
		
		Класс предназначен для работы с JSON-файлами.
		
			Выполняет следующие функции:
		
		1. Считывает из файлов документы для поиска, указанных в config.json.
		2. Считывает запросы из файла requests.json.
		3. Считывает значение максимального количества ответов на запрос из файла config.json.
		4. Записывает в файл answers.json результаты поиска.
		
			Класс Documents
		
		Класс реализует загрузку и хранение базы документов.
		
			Выполняет следующие функции:
		
		1. Обновляет список документов, по которому будет совершаться поиск.
		2. Хранит список документов.
		3. Предоставляет ссылку на список документов.
		
			Класс Requests
		
		Класс реализует загрузку и хранение базы запросов.
		
			Выполняет следующие функции:
		
		1. Обновляет список запросов, по которому будет совершаться поиск.
		2. Хранит список запросов.
		3. Предоставляет ссылку на список запросов.
		
		
		
			Класс InvertedIndex
		
		Класс реализует инвертированную индексацию документов.
		
		1. Индексирует слова в документах.
		2. Хранит базу инвертированного индекса.
		3. Предоставляет ссылку на базу инвертированного индекса.
		
			Класс RelevantResponse
		
		Класс определяет релевантность ответов.
		
			Выполняет следующие функции:
		
		1. Рассчитывает абсолютную и относительную релевантность ответов по документам.
		2. Хранит базу релевантных ответов.
		3. Предоставляет ссылку на базу релевантных ответов.
