## Аргументы программы
Используются следующие аргументы (в скобках указаны значения по умолчанию):

/c &nbsp Путь файла конфигурации (config.json)\
/r    Путь файла запросов (requests.json)\
/a    Путь файла ответов (answers.json)\
/l    Путь файла конфигурации логирования (logger.json)\
/m    Путь файла конфигурации очереди сообщений (messageQueue.json)\
/lc   Логировать события в консоль (yes)\
/p    Количество знаков после запятой (6)\
/f    Ширина вывода (2)\
/t    Желаемое количество потоков (количество ядер минус два)\
/e    Максимальное количество непрочитанных файлов (1)\
/?    Справка\

Использование очевидно и не требует пояснений.
